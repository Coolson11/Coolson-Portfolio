# Coolson Portfolio

A modern, professional portfolio website built with HTML, CSS, JavaScript, and Bootstrap.

## Features

- **Modern Design**: Clean, responsive layout with a professional look
- **Animations**: Smooth scroll animations using AOS (Animate On Scroll)
- **Particle Effects**: Interactive background particles using Particles.js
- **Responsive**: Fully responsive design that works on all devices
- **Bootstrap Integration**: Utilizes Bootstrap 5 for perfect UI/UX
- **Interactive Elements**: Hover effects, smooth scrolling, and form handling

## Sections

- **Home**: Hero section with particles background
- **About**: Personal introduction and profile picture
- **Skills**: Technical skills with progress bars
- **Projects**: Showcase of key projects
- **Contact**: Contact form and information

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Particles.js
- AOS (Animate On Scroll)
- Font Awesome Icons

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. For local development, run a local server (e.g., `python -m http.server 8000`)

## Customization

- **Personal Information**: Update the content in `index.html`
- **Styling**: Modify `css/styles.css` for custom styles
- **Functionality**: Edit `js/script.js` for additional JavaScript features
- **Images**: Replace placeholder images in the `assets` folder

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
